# Use Langfuse

## What it is

Langfuse is an open-source LLM observability and analytics platform. It helps developers monitor, debug, and improve applications that use large language models.


1. Tracing & Logging : Captures detailed traces of each LLM request, shows inputs, outputs, latencies, token usage, and metadata

2. Prompt & Model Analytics: Tracks how different prompts or models perform, compares cost, latency, and quality

3. Evaluation Tools: Supports automated and manual evaluation of model outputs


## Use it

We host our own instance of langfuse : https://langfuse.lab.sspcloud.fr/ where you can create your organisation and projects.

1. **Connecting to the Platform** :

    - Access the SSP Cloud platform via your browser.
    - Log in with your SSO credentials.

2. **Generating the API Key** :

    - Once logged in, go to the project settings.
    - Then go to the API KEYS section
    - There is a button dedicated to the creation of new api keys.

3. **Securing the Key** :
    - Make sure to note your secret key correctly, you won't be able to retrieve it later.
    - Do not share it publicly to avoid unauthorized access.
